# 工业互联网平台测试用例

## 1. 测试目标

本测试用例用于验证工业互联网平台当前版本的功能完整性、核心业务闭环、接口可用性、构建结果和交付质量。测试重点为登录、设备管理、模拟上报、规则告警、历史数据、告警处置、任务启停和日志审计。

## 2. 测试环境

| 项目 | 说明 |
| --- | --- |
| 后端 | Spring Boot 3 微服务；网关 8080、核心服务 8081、可视化视频服务 8082 |
| 前端 | Vue3 + Vite，端口 5173 |
| 默认数据库 | H2 文件数据库 |
| MySQL 脚本 | MySQL 8.x 兼容脚本 |
| 默认账号 | `admin / 123456` |
| 默认 Token | `panda-iot-demo-token` |

## 12. 验证测试结论

项目核心功能测试通过，后端微服务和前端均可构建，核心业务闭环可真实运行。服务管理、可视化大屏配置与发布、视频设备台账、流代理配置、视频广场通道状态等能力达到当前版本验证要求。后续生产化改进可增加自动化测试、真实设备接入、标准权限、真实流媒体播放、GB28181 级联和持续集成。
