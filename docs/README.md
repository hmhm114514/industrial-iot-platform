# 工业互联网平台开发实践文档目录

本目录为《工业互联网平台开发实践》课程项目配套文档，内容围绕 `industrial-iot-platform` 项目编写。项目采用 Spring Boot 3 + Vue3 前后端分离架构，默认使用 H2 文件数据库运行，同时提供 MySQL 建表和示例数据脚本用于课程交付。

## 推荐阅读顺序

1. `需求分析报告.md`：说明项目背景、用户角色、功能需求和验收目标。
2. `总体规划报告.md`：说明建设目标、技术路线、模块规划、实施计划和风险控制。
3. `架构设计报告.md`：说明前后端分离架构、后端分层、数据库、鉴权和核心流程。
4. `平台设计报告.md`：逐模块说明页面、业务流程和数据交互设计。
5. `项目报告.md`：作为课程提交主体报告，综合描述需求、设计、实现、测试和总结。
6. `API清单.md`：列出核心 REST API。
7. `数据库设计.md`：说明主要数据表、字段和 H2/MySQL 策略。
8. `测试用例.md`：列出功能、接口、异常、构建和验收测试用例。
9. `验收矩阵.md`：把实践指导书和 PandaX 参考功能映射到本项目实现。
10. `周报.md`：记录课程实践过程。
11. `视频介绍与答辩提纲.md`：用于录制演示视频和答辩准备。

## 项目范围说明

本课程项目参考 PandaX 物联网平台功能，采用“完整可运行原型 + 核心闭环真实实现 + 扩展模块演示化实现”的方式控制范围。

- 核心真实闭环：登录认证、产品/设备管理、设备模拟上报、规则告警、历史数据、历史告警处置、仪表板统计、日志审计。
- 扩展演示模块：服务管理、解析脚本、组态大屏、视频中心、任务中心、固件管理、系统日志等。
- 文档说明模块：AI 智能体、开发工具等 PandaX 扩展能力在报告中说明其设计思路和扩展方向。

## 运行环境摘要

- 后端：Java 17 目标版本，Spring Boot 3，Maven。
- 前端：Vue 3，Vite，Element Plus，ECharts。
- 默认数据库：H2 文件数据库，JDBC URL 为 `jdbc:h2:file:./data/iot-platform-db`。
- MySQL 交付：`database/schema-mysql.sql` 与 `database/data-mysql.sql`。
- 默认账号：`admin / 123456`。
