# 工业互联网平台验收矩阵

## 1. 验收说明

本矩阵将实践指导书要求和 PandaX 参考功能点映射到当前项目的实现位置、页面/API 和验证方式。项目采用“核心闭环真实实现 + 扩展模块演示化实现”的方式完成课程交付。

## 3. 核心闭环验收路径

| 步骤 | 操作 | 预期结果 |
| --- | --- | --- |
| 1 | 启动后端和前端 | 后端 8080、前端 5173 可访问 |
| 2 | 使用 `admin/123456` 登录 | 登录成功进入仪表板 |
| 3 | 进入设备管理查看设备 | 可看到“注塑机温度传感器01”等示例设备 |
| 4 | 模拟上报温度 `88.6` | 接口返回遥测数据和告警列表 |
| 5 | 查看历史数据 | 出现最新遥测记录 |
| 6 | 查看历史告警 | 出现 HIGH 等级 OPEN 告警 |
| 7 | 处置告警 | 告警状态变为 CLOSED |
| 8 | 查看操作日志 | 出现模拟上报和处置告警日志 |
| 9 | 关闭规则后再次高温上报 | 不再新增告警 |

## 4. 非阻断说明

- 视频中心和组态大屏为课程演示化实现，未接入真实视频流和拖拽设计器。
- 设备接入采用模拟上报，未接入真实 MQTT Broker。
- 鉴权采用简化 Token，后续可扩展为 JWT 和 RBAC。
- 前端构建包体积较大提示不影响课程运行和验收。
